Source Code Repository

Your complete source code must be hosted in a Git repository and accessible for review. The repository must include all of the following:

  • Full application source code
  • Dockerfile for containerising your application
  • All CI/CD configuration files (e.g., .github/workflows/*.yml)
Notice: The name for your project's folder will contains your team name and project name.
For example:
  • team: 30
  • project: urlservice
Your project's folder is 30_urlservice
Important: All three deliverables are mandatory. Incomplete submissions may result in grade deductions. Ensure your Git repository is set to public or that access has been shared with your instructor before the deadline.
