AES-ECB

Create the aes_ecb.py file. This file will encrypt the tux.bmp file according to the AES-ECB algorithm. 

sudo nano aes_ecb.py

Copy the code below to this file

from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad

key = b"aaaabbbbccccdddd"
cipher = AES.new(key, AES.MODE_ECB)

with open("./Downloads/tux.png", "rb") as f:
        clear = f.read()
        clear_trimmed = clear[54:-12]
	  padded_text = pad(clear_trimmed, AES.block_size)

        ciphertext = cipher.encrypt(padded_text)
        ciphertext = clear[0:54] + ciphertext + clear[-12:]
with open("./Downloads/tux_ecb.bmp", "wb") as f:
        f.write(ciphertext)

Execute aes_ecb.py:

./lab03_04/bin/python aes_ecb.py

Check the encrypted file tux_ecb.bmp in the Downloads folder.

AES-CBC

Create the aes_cbc.py file.

sudo nano aes_cbc.py

Execute aes_cbc.py:

./lab03_04/bin/python aes_cbc.py
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad
key = b"aaaabbbbccccdddd"
iv = b"1111222233334444"
cipher = AES.new(key, AES.MODE_CBC, iv)

with open("./Downloads/tux.png", "rb") as f:
        clear = f.read()
        clear_trimmed = clear[54:-12]
        padded_text = pad(clear_trimmed, AES.block_size)
        ciphertext = cipher.encrypt(padded_text)
        ciphertext = clear[0:54] + ciphertext + clear[-12:]
with open("./Downloads/tux_cbc.bmp", "wb") as f:
        f.write(ciphertext)
